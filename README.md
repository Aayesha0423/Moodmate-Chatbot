# MoodMate: Your AI Companion Chatbot
MoodMate is a friendly AI chatbot built with Python and Gradio.  
It interacts based on your mood, comforting you when you're sad and celebrating when you're happy.  

## Features
- Mood-focused conversational assistant  
- Compatible with Gradio 4.x and 5.x  
- Uses Groq API (Llama 3.1-8b)
